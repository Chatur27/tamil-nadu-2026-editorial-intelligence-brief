# AtliQ Editorial Control Room — Tamil Nadu 2026

Open `index.html` in any modern browser. This is a static dashboard supporting the PowerPoint stakeholder deck.

## Core storyline
- Scale: 163 of 234 constituencies changed winning party.
- Geography: Chennai Metro had 30 of 32 seats change winning party.
- Competitiveness: 103 constituencies were won by less than 5 percentage points.

## Role in submission
This dashboard is a supporting tool. The main submission narrative is the PowerPoint deck and the 5–7 minute video walkthrough.

## Neutrality discipline
The dashboard reports result patterns only. It does not infer voter motives, vote transfers, causes, or future outcomes.
